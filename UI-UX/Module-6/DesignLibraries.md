
## Learning Objectives
1. Understand the concept and benefits of design libraries in Figma.  
2. Develop skills in creating and organizing design assets within a library.  
3. Practice efficient design workflows using library components.  
4. Enhance collaboration and consistency in design projects.  

---

## Activity Outlines

### 1. Design Library Creation
- Create a new Figma file and set up a design library  
- Create components for common design elements such as:
  - Buttons  
  - Icons  
  - Headers  
  - Footers  
- Apply principles of:
  - Typography  
  - Color theory  
  - Spacing  
- Emphasize proper naming conventions and clear organization within the library  

---

### 2. Design Challenge
- Design a scheduling interface for an airport flight booking system  
- Use the components created in your design library  

---

## Additional Tips
- Create a reasonable number of screens  
- Ensure all relevant app requirements are captured  
- Maintain consistency across designs using your library components
