
## Learning Objectives
- Understand the concept and benefits of design libraries in Figma  
- Develop skills in creating and organizing design assets within a library  
- Practice efficient design workflows using library components  
- Enhance collaboration and consistency in design projects  
- Publish and maintain design libraries  

---

## Activity Outlines

### 1. Publish Design System
- Publish your design system from the previous project  

---

### 2. Design Challenge

#### a. Food Delivery System
Design a food delivery application that allows users to:
- Order their favorite food  
- Track the delivery process until it is received  

#### b. Use of Design Library
- Build the design using your published component library  

---

## NB
- Add any missing components needed into your design library  
- Republish the library to make new components available for use  

---

## Additional Tips
- Create a reasonable number of screens  
- Ensure all relevant app requirements are captured  
- Maintain consistency by using your design library components
